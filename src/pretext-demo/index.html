Pretext POC @chenglou/pretext

DOM-free text measurement and layout by Cheng Lou (Midjourney). Measures text ~500x faster than getBoundingClientRect() by using canvas.measureText() with cached segment data.

1. Text Height Measurement

Enter text and a container width. Pretext measures the wrapped height without touching the DOM.

2. Canvas Text Rendering

Pretext computes line breaks, then we render each line on a <canvas>. Drag the right edge to resize — re-layout runs in pure arithmetic (~0.0002ms per block).
